KerbalAPI

In its 0.14 release, Kerbal Space Program added a class loader, enabling us to write custom functionality into the game. Yay! Unfortunately, KSP is written on top of Unity, and there's very little separating the two. The KSP classes are built in the Unity style, with obscure interactions between class fields.

The goal of KerbalAPI is to roll as much functionality as possible into a dedicated API that does not sit on top of Unity.

$ git checkout git@github.com:KerbalAPI/KerbalAPI.git
